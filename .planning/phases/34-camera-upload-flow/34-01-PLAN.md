Create `frontend/src/pages/sites/CameraUploadFlow.tsx` — a component that provides a direct camera/gallery picker flow separate from CreateNoteModal.

**Component structure:**

```tsx
interface CameraUploadFlowProps {
  open: boolean
  onOpenChange: (open: boolean) => void
  siteId: string
  onSuccess: () => void
}
```

**UX flow:**
1. When `open` becomes true, immediately trigger a hidden `<input type="file" accept="image/*" capture="environment">` click to open the native camera/gallery picker
2. After file selection, show a Dialog with:
   - Image preview (using `URL.createObjectURL` for the selected file)
   - Optional textarea labeled "Notiz hinzufügen (optional)" with placeholder "Beschreibung zum Foto..."
   - "Abbrechen" and "Hochladen" buttons
3. On submit:
   - If online: call `useUploadSitePhoto()` → get `photo_url` → call `useCreateActivity()` with `{ activity_type: "photo", content: trimmedNote || undefined, photo_url }`
   - If offline: call `queuePhotoUploadAction({ siteId, file, content: trimmedNote || undefined })`
4. On success: show toast "Foto hinzugefügt" (or "Foto offline gespeichert" for offline), call `onSuccess()`, close dialog
5. On error: show toast error "Foto konnte nicht hochgeladen werden"
6. On cancel/close: revoke object URL, clear state

**Implementation details from existing patterns (follow CreateNoteModal.tsx):**
- Use shadcn/ui `Dialog`, `DialogContent`, `DialogHeader`, `DialogTitle`, `DialogFooter`
- Use `useUploadSitePhoto()` and `useCreateActivity()` from `@/lib/api/hooks`
- Use `isOnline()` from `@/lib/offline/sync` for online/offline check
- Use `queuePhotoUploadAction` from `@/lib/offline/queue` for offline uploads
- Use `toast` from `sonner` for success/error notifications
- Use `Button` and `Textarea` from shadcn/ui
- Track `isPending` status from both mutations to disable submit button during upload

**Hidden file input technique:**
- Use a `useRef` for the hidden file input element
- Use `useEffect` that triggers `inputRef.current?.click()` when `open` changes to `true`
- Set `accept="image/*"` and `capture="environment"` on the input
- On file selection, store the file in state and show the dialog with preview

**Important:** Do NOT modify CreateNoteModal.tsx — it will be reworked in Phase 35 (Document Upload Rework).
