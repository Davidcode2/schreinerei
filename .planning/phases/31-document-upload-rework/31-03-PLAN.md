## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|

| T-31-11 | T | filename display | mitigate | Render filenames as plain text nodes only; do not inject HTML from attachment metadata. |
| T-31-12 | D | preview error path | mitigate | Fallback to a static shell and text when blob fetch fails so one bad preview does not break the full feed render. |
